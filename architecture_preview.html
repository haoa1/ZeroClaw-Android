Architecture Overview

UI Layer (Kotlin/Compose)

  • Dashboard, Agents, Terminal screens
  • Navigation with ZeroClawNavHost
  • ViewModels for state management
  • Theme system with Material 3

Android Services & Data

  • Foreground service for 24/7 operation
  • Room database with 7+ DAOs
  • EncryptedSharedPreferences (AES-256-GCM)
  • WebSocket channels & OAuth

FFI Bridge (UniFFI)

  • 34 FFI functions Kotlin↔Rust
  • Automatic type marshaling
  • Error handling with catch_unwind
  • Generated Kotlin bindings

Rust Core Engine

  • AI Agent Gateway & conversation management
  • Rhai scripting engine (Terminal REPL)
  • 32+ AI provider integrations
  • Plugin system with tools

External Integrations

  • 32+ AI providers (OpenAI, Claude, etc.)
  • Plugin system extensions
  • Android system services
  • Hardware-backed security

Build System

  • Gradle Kotlin DSL
  • Gobley plugin for Rust compilation
  • cargo-ndk for cross-compilation
  • UniFFI Kotlin bindings generation

Key Architectural Features:

  • Performance: Rust core provides native-speed AI processing
  • Safety: Memory safety in Rust, type safety in Kotlin
  • Security: Hardware-backed encryption via Android Keystore
  • Battery Optimization: Foreground service with START_STICKY
  • Extensibility: Plugin system for adding new capabilities
  • Developer Experience: REPL interface for power users
